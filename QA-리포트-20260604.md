## 4. 오늘 발견·수정한 핵심 버그 (시간순)

1. **HTML 구조**: 대시보드 `</div>` 2개 누락 → 12개 페이지가 대시보드 안에 갇혀 빈 화면 → 수정, 13페이지 전부 복구
2. **실시간 렌더 폭주**: 장중 호가 틱마다 호가창 전체 재렌더 → 브라우저 멈춤 → 가격 5회/초·호가 3.3회/초 스로틀
3. **토큰 동시 발급 충돌**: 만료 순간 동시 재발급 → 전부 실패 → 발급 1건 공유 + 65초 쿨다운
4. **주문 큐 우회**: 엔진 스캔 중 주문 시 "초당 거래건수 초과" → 주문도 큐(high) 경유 + 자동 재시도(kisPost)
5. **동기 파일쓰기 블로킹**: 로그/상태 writeFileSync가 파일 잠금에 걸리면 서버 전체 정지 → 비동기+디바운스 전환
6. **KIS 무응답 큐 정체**: 타임아웃 없는 호출이 큐를 영구 정지 → 10초 타임아웃
7. **큐 적체 발산**: KIS 느려질 때 수요>처리량 → 백프레셔(low 10건/high 25건 상한 + 즉시 실패→폴백)
8. **브라우저 연결 고갈**: 무응답 fetch가 연결풀 점유 → 12초 타임아웃
9. **취소 주문 ORD_UNPR 누락** → 수정
